Quick summary of this project

The project is now deployed at yorubasystems.com. The orginal scope of this project was for a simple, online vocabulary quiz.

This page is supposed to document the interesting parts of my research and the software development process.

The technical challenge was to build this web application using only core programming language APIs, so without utilising the Frameworks and libraries that are abundant for JavaScript, PHP and CSS. For the developer, the benefit of this approach was improved awareness of those core APIs. Drawbacks of this approach, would apparently include speed of development (the implementation phase) and that the code may be less maintainable in future if the application grows beyond it's current limited scope.
